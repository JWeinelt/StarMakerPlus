# ⭐ StarMakerPlus
This is an easy-to-use editor for the **StarMaker GalactiCraft Addon.**
It provides a simple interface for creating and editing star systems.

## 🌟 Highlights
- ✔ Easy to use
- 💻 On-screen instruction
- 📁 No installation required
- 📝 Graphical interface for editing celestial json files
- 📥 Lightweight and fast

## 🎮 Current features supported
- [X] Creating **solar systems**
- [X] Creating **stars**
- [ ] Managing **galaxies**
- [X] Creating **planets**
- [ ] Creating **moons**
- [ ] Creating **asteroids**
- [ ] Creating **satellites**
- [ ] Creating **new biomes for planets**

*Some features may be limited by the capabilities of the **StarMaker Galacticraft Addon.***

## 📥 How to use
> [!NOTE]
> This **is not a mod/addon**. It is a simple editor for creating "celestialpacks" for the **StarMaker GalactiCraft Addon.**

1. Download the latest release from the [releases page](https://github.com/JWeinelt/StarMakerPlus/releases)
2. Copy the downloaded file (exe/jar) to any folder you want
3. Launch it using a double click

> [!TIP]
> StarMakerPlus saves all files in ``~/SMPlusProjects`` and in ``APPDATA/StarMakerPlus``.
> You can change the location of the project base folder in the settings.

### 🎮 Preparing Minecraft for testing
It's recommended to install a small modpack just with essential mods for testing, as for now,
every change requires a restart of the game. This may take quite long with big modpacks.

Please head to the [wiki](https://docs.starmakerplus.julianweinelt.de/testing) for more information.
